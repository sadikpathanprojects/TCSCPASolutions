TCS_xplore_java_hands_on_2021 | Java iteration hands on 1:Finding the lowest valued charachter in the string
Write the main method in solution class. the method will take the string as input and print the minimum valued charachter in that string (as per ascii value) Input: HellO Output:H As H is having the lowest ascii value.

Code to the above Question is given below.

import java.util.Scanner; public class solution { public static void main(String[] args) { Scanner sc=new Scanner(System.in); String a=sc.nextLine(); int temp=(int)a.charAt(0); for (int i = 1; i <a.length() ; i++) { if((int)a.charAt(i)<temp) { temp=(int)a.charAt(i); } } System.out.println((char)temp); } }
